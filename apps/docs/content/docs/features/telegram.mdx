# Human-in-the-Loop Workflow

AGIAAS leverages the powerful [messaging patterns](https://hermes-agent.nousresearch.com/docs/user-guide/messaging) established by the **Hermes Agent**. Safety is prioritized through a secure, mobile-first human-in-the-loop (HITL) system.

## The Hermes HITL Pattern

1. **Incident Triggered**: A webhook notifies the @agiaas/webhook-server.
2. **Hermes Analysis**: The foundation agent analyzes the incident.
3. **Telegram Notification**: You receive an interactive message via the [Hermes-inspired Gateway](https://hermes-agent.nousresearch.com/docs/user-guide/messaging):
   - **Context**: A summary of what went wrong based on codebase memory.
   - **Proposed Fix**: A clean diff generated by the Hermes code skills.
   - **Reasoning**: A breakdown of the AI's logical process.
   - **Cost Index**: Real-time token usage estimate.
4. **Interactive Commands**: You interact with the agent using the same intuitive flow as the core [Hermes Agent](https://hermes-agent.nousresearch.com/docs/):
   - ✅ **Approve**: Agent pushes the fix and opens a documented PR.
   - ❌ **Reject**: Agent stops, logs the rejection, and learns from the feedback.
   - 💬 **Re-Analyze**: Request a deeper dive or a specific fix strategy.

## Messaging Gateway Setup

Follow the [Hermes Agent Messaging Guide](https://hermes-agent.nousresearch.com/docs/user-guide/messaging) for creating your bot:

1. Create a bot via [@BotFather](https://t.me/botfather).
2. Note your `TELEGRAM_BOT_TOKEN`.
3. Locate your `CHAT_ID` using [@userinfobot](https://t.me/userinfobot).
4. Add these to your root `.env` configuration.

## Why the Hermes Flow?

- **Remote Action**: No need to SSH or be at your laptop—manage incidents from anywhere.
- **Instant Triage**: Get high-signal alerts that help you make decisions in seconds.
- **Secure Control**: The gateway ensures that only authorized users can trigger autonomous actions.

> [!TIP]
> This messaging flow is a direct implementation of the [Hermes Messaging Gateway](https://hermes-agent.nousresearch.com/docs/user-guide/messaging) philosophy: "Control your agent from the devices you already use."
