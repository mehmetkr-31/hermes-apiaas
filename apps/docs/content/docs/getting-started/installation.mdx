# Installation

Follow these steps to get AGIAAS up and running.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js**: v18 or higher (Recommended: v20)
- **pnpm**: v9 or higher
- **Python**: v3.12 or higher
- **GitHub CLI (`gh`)**: For repository interactions.
- **SQLite3**: For the local database.
- **cloudflared**: For webhook tunnels.

## Step 1: Clone the Repository

```bash
git clone https://github.com/mehmetkr-31/agiaas.git
cd agiaas
```

## Step 2: Install Dependencies

AGIAAS uses `pnpm` for managing the monorepo.

```bash
pnpm install
```

## Step 3: Environment Configuration

Instead of manually copying files, use the interactive setup tool:

```bash
pnpm setup
```

This script will:
- Generate a 32-character `DB_ENCRYPTION_KEY` for securing your credentials.
- Generate a `BETTER_AUTH_SECRET` for dashboard security.
- Prompt for your AI Provider keys (Nous Research or OpenRouter).
- Configure your local GitHub CLI paths.

## Step 4: Run with Docker (Recommended)

The easiest way to run the entire stack (Dashboard + API + Agent) is using Docker:

```bash
docker-compose up --build
```

The Web Dashboard will be available at [http://localhost:3678](http://localhost:3678).

## Step 5: Local Development (Manual)

If you prefer to run services individually without Docker:

1. **Migrate the Database**:
   ```bash
   pnpm db:migrate
   ```

2. **Install Python Agent Dependencies**:
   ```bash
   pnpm python:install
   ```

3. **Start the Services**:
   ```bash
   pnpm dev
   ```

> [!IMPORTANT]
> The default port for the dashboard in local development is **3678**.

> [!NOTE]
> The webhook server will automatically open a Cloudflare tunnel and update your GitHub repository webhook settings.
