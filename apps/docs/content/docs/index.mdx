# AGIAAS

**AGIAAS** (Autonomous GitHub Incident Analysis & Automation System) is an **Intelligent Agent Platform** that monitors your GitHub infrastructure, analyzes anomalies using state-of-the-art AI, and facilitates automated fixes via a Telegram-integrated **human-in-the-loop** workflow.

> Think of it as an on-call SRE that watches your repos 24/7 — triages incidents, writes patches, opens PRs, and waits for your approval before merging. All from your phone.

## Built with Hermes Agent

AGIAAS is built on the [Hermes Agent](https://hermes-agent.nousresearch.com/docs/) paradigm by **Nous Research**. We leverage the self-improving autonomous engine to handle complex coding and debugging tasks — it's not a tethered copilot, it's a fully autonomous agent.

## The 6-Step Pipeline

When an incident hits, AGIAAS runs an autonomous pipeline:

1. **Monitor** — GitHub webhook fires (issue, PR fail, CI error)
2. **Reason** — Mixture-of-Agents debate for root-cause analysis
3. **Research** — Web search + session history for context
4. **Remediate** — Parallel task execution (patch, restart, cache)
5. **Report** — Telegram UI Card with diff, cost, and approval buttons
6. **Learn** — Auto-generated runbook for future incidents

## Key Features

- **🧠 Multi-Agent Reasoning**: Mixture-of-Agents debate for deep root-cause analysis
- **🔀 Dynamic Model Selection**: Per-repo configurable models (Hermes-4-405B, Claude-3.5-Sonnet, Gemini Flash) with auto-fallback
- **🛠️ Autonomous Patching**: Full codebase exploration, code patching, and PR generation
- **📱 Mobile Approval**: Approve or reject AI actions via structured Telegram UI Cards
- **📡 Multi-Provider Webhooks**: GitHub, Sentry, Vercel, and Cloudflare event ingestion
- **💰 Token Economics**: Dynamic cost calculation based on actual model usage via OpenRouter
- **📓 Auto Runbooks**: Every resolved incident generates a runbook for future reference
- **🔐 AES-256-GCM Encryption**: All secrets encrypted at rest
- **⏪ CI/CD Rollback**: One-tap rollback to the last successful workflow run
